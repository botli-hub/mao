# MAO 平台 — 原始需求与原型文档

本目录存放 MAO 平台立项阶段的原始需求文档与交互原型，是整个系统设计的输入来源。

---

## 文件说明

| 文件名 | 说明 |
|---|---|
| `product_requirements.txt` | 产品需求说明文档，描述 MAO 平台的整体定位、核心功能与业务场景 |
| `b_side_prototype.html` | **B 端管理控制台**交互原型（HTML 可视化），包含智能体工厂、技能注册中心、全局编排画布、监控调度大盘等模块 |
| `c_side_prototype.html` | **C 端 LUI 统一工作站**交互原型（HTML 可视化），包含自然语言对话、GUI 卡片交互、AI 托管大盘等模块 |
| `api_reference_prototype.html` | **API 接口参考原型**（HTML 可视化），为系统设计阶段的接口规范参考，最终接口文档见 `../04_api/api_reference.md` |

> **使用方式**：直接在浏览器中打开 `.html` 文件即可预览交互原型。
