MAO 全景 OpenAPI (资深研发终审版)

v9.0.0-PROD
Base URL: /api/v1

接口 I/O 深度校验与状态机闭环

本文档已经过资深研发 Review。全量补充了高并发场景下的数据防线，包括：Cron 的防重叠队列、Webhook 的防重放签名、GUI 卡片的同步/异步双态响应，以及 DAG 画布的条件分支逻辑。
